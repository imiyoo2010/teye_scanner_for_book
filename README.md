## 内容介绍
> 由于本书并不是按照代码逻辑来进行编写的，所以建议读者先将项目代码运行起来后，然后再对照书籍来进行学习；代码已作处理，只保留完整的扫描器核心功能，主要以学习和研究为目的，相信认真读完本书后是有能力自己来实现完整云扫描功能；如果您在学习中有遇到问题或不清楚的地方，可以就具体问题进行留言交流和探讨。

 > python teye.py
